# Day 27: Hunting Techniques

## 🧠 Key Concepts
* **Analyzing Network Traffic:** Spotting foreign IPs, unusual ports, or odd-hour traffic.
* **Analyzing Endpoint Logs:** Investigating strange PowerShell commands or new, unknown processes.
* **IOCs (Indicators of Compromise):** Searching for malicious IP addresses, domains, file hashes, or filenames.
* **Hunting Hypothesis:** An educated guess ("I suspect an attacker is moving laterally via PowerShell") that drives the investigation.
