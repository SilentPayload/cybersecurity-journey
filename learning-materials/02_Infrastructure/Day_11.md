# Day 11: Cryptography Basics

## 🧠 Key Concepts


[Image of symmetric vs asymmetric encryption]


* **Symmetric Encryption (AES):** Same key for encryption/decryption (Private key cryptography).
* **Asymmetric Encryption (RSA):** Public/Private key pairs.
* **Hashing (SHA-256):** One-way function for data integrity (fixed-size string).
* **Digital Certificates:** "Electronic passport" issued by a CA (Certificate Authority) to prove identity.
* **Real-World Use:** SSL/TLS for secure web browsing (HTTPS).
