# Day 7: OSI Model & Network Components

## 🎯 Learning Objectives
Understand how data moves across a network and the role of key hardware components.

## 🧠 Key Concepts


* **Networking Basics:** Networks connect devices to share resources. 
* **Addressing:**
    * **IP Address:** Identifies a host (IPv4/IPv6).
    * **MAC Address:** Physical "fingerprint" of the device.
* **Hardware:**
    * **Switch:** Connects devices in a LAN.
    * **Router:** Connects different networks (e.g., LAN to WAN).
* **OSI Model (The Framework):**
    1. Physical (Cables)
    2. Data Link (MAC)
    3. Network (IP)
    4. Transport (TCP/UDP)
    5. Session
    6. Presentation
    7. Application (HTTP/HTTPS)
