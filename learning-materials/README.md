# Teaching Materials

Scripts, tools, and lesson plans built for cybersecurity students.

## Structure
- `python/` — Python scripts built in class
- `linux/` — Linux lab challenges
- `wireshark/` — Packet capture labs
- `forensics/` — Autopsy and digital forensics labs

## Modules Covered
- Linux fundamentals
- Networking and Wireshark
- Python for security
- SIEM and Splunk
- Digital forensics
- Web application security
- Penetration testing
