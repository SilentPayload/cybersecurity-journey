# Day 8: Network Architecture & Common Attacks

## 🧠 Key Concepts
### Common Attacks
* **Sniffing:** Capturing data packets (Passive vs. Active).
* **MITM (Man-in-the-Middle):** Intercepting/modifying traffic between parties.


### Defensive Measures
* **IDS:** Monitors/Alerts on suspicious activity.
* **IPS:** Blocks malicious traffic in real-time.
* **Segmentation:** Splitting networks (VLANs) to limit access.
* **Firewalls:** Controls traffic via rules (IP, Port, Protocol).
* **VPN:** Creates an encrypted tunnel for data.
