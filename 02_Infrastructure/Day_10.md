# Day 10: OS Security

## 🧠 Key Concepts
Hardening systems is the first line of defense against local and remote exploits.

* **Linux Security:** * User/Group management.
    * Sudo/root permissions.
    * SSH key-based authentication.
* **Windows Security:**
    * NTFS permissions (user access).
    * Event Viewer logs (monitoring).
