# Day 13: Python Basics (Variables & Strings)

## 🧠 Key Concepts
* **Syntax:** Clean, readable indentation is part of Python's power.
* **Variables:** Containers for storing data values.
* **Data Types:** Understanding strings, integers, floats, and booleans.
* **String Formatting:**
    * **F-Strings:** The modern standard (`f"Hello {name}"`).
    * **Concatenation:** Using `+` to join strings/variables.
