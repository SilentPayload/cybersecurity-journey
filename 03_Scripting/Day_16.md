# Day 16: Modules & Conditionals

## 🧠 Key Concepts
* **Modules:** Files containing reusable code (e.g., `math`, `hashlib`, `socket`, `scapy`).
* **Conditional Statements:** `if`, `elif`, `else` determine logic flow based on conditions.
