# Day 14: Python Basics (Input & Operators)

## 🧠 Key Concepts
* **Data Type Conversion:**
    * **Implicit:** Python handles it automatically.
    * **Explicit:** Manual casting (e.g., `int("10")`).
* **Input Function:** `input()` collects data from the user.
* **Operators:** Special symbols that perform operations.
    * **Arithmetic:** `+`, `-`, `*`, `/`.
    * **Comparison:** `==`, `!=`, `>`, `<`.
    * **Logical:** `and`, `or`, `not`.
    * **Membership:** `in`, `not in`.
