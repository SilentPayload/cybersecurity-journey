# Day 32: Web & Cloud Security

## 🧠 Key Concepts
* **Web Security:** Understanding the Client-Server architecture.


[Image of client-server web application architecture]

* **Common Vulnerabilities:**
    * **SQL Injection (SQLi):** Manipulating database queries.
    * **Cross-Site Scripting (XSS):** Injecting malicious scripts into web pages.
* **Cloud Security:**
    * **SECaaS (Security as a Service):** Outsourcing security tasks.
    * **XaaS (Anything as a Service):** NaaS, DaaS, BaaS.
    * **Shared Responsibility Model:** Understanding what you vs. the cloud provider secures.
    * **Cloud Risks:** Publicly exposed S3 buckets, weak IAM policies, and insecure APIs.
